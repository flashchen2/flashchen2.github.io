三、Linux操作系统

  • 学习链接：https://www.bilibili.com/video/BV1pE411C7ho/?

1. bash是什么

  1. bash是一个命令处理器，运行在文本窗口中，并能执行用户直接输入的命令。即终端
  2. bash还能从.sh文件中读取linux命令，称之为脚本
  3. bash还支持通配符、管道、命令替换、条件判断等逻辑控制语句

2. 目录操作命令

1. ls - 显示目录下的内容

  • 英文原意：list
  1. 以列表形式显示文件：ls -l
  2. 显示隐藏文件：ls -a

2. pwd - 查看当前目录

  • 英文原意：print work directory

3. cd - 切换所在目录

  • 英文原意：change directory
  1. 当前目录：cd .
  2. 返回上一级目录：cd ..
  3. 返回根目录：cd ~

4. mkdir - 创建空目录

  • 英文原意：make directory
  • 注：只能一级一级创建文件夹，不能同时创建多级目录
  1. 创建文件夹：mkdir 目录名，如mkdir aa

3. 文件操作命令

1. touch - 创建空文件

  1. 创建文本：touch text01.txt
  2. 创建隐藏文件：touch .text02.txt

2. cat - 查看文件内容

  1. 查看文件内容：cat 文件名
  2. 查看文件内容，并显示行号：cat -n 文件名

3. more - 分屏显示文件内容

  • 一般当文件内容比较多的时候，使文件的内容在终端分页进行打印
  1. 查看文件内容：more 文件名

4. 目录 & 文件操作命令

1. rm - 删除目录或文件

  • 英文原意：remove
  1. 移除文件：rm text01.txt
  2. 移除文件夹：rm -d aa 或 rm -r aa

2. cp - 复制文件

  • 英文原意：copy
  1. 复制文件：cp 文件名 复制的位置， cp test01.txt aa
  2. 复制文件夹：cp -r 文件夹名 复制的位置， cp -r aa/bb .

3. mv - 剪切文件或目录

  • 英文原意：move
  1. 移动文件位置：mv 源文件 目标目录，mv test01.txt aa
  2. 移动文件夹位置：mv 源目录 目标目录，mv bb aa

5. 搜索文件内容

1. find - 制定目录下查找文件

  1. 精确搜索 文件所在位置：find 搜索路径 -name 查找的文件名，find /home -name "test02.txt"
  2. 模糊搜索 文件所在位置：find 搜索路径 -name 残缺的名字*或find 搜索路径 -name 残缺的名字？？

​ 例：find /home -name test* 或 find /home -name test??????

  1. 不区分文件大小写的搜索：find 搜索路径 -iname 查找的文件名

2. locate - 按照文件名搜索文件

  • 解释：locate命令其实是find -name的另一种写法，但是要比后者快得多，原因在于它不搜索具体目录(find 是去硬盘找)，而是搜索一个数据库/var/lib/mlocate/mlocate.db，这个数据库中含有本地所有文件信息。Linux系统自动创建这个数据库，并且每天自动更新一次，所以使用locate命令查不到最新变动过的文件，为了避免这种情况，可以在使用locate之前，先使用updatedb命令手动更新数据库
  • 优点：搜索速度快
  • 缺点：只能按照文件名来搜索文件，而不能执行更复杂的搜索
  1. 查找文件所在位置：locate 查找的文件名

3. grep - 提取文本中所需的信息

  1. 搜索文件内容：grep 搜索的内容 文件名
  2. 搜索文件内容，忽略大小写：grep -i 搜索的内容 文件名
  3. 搜索文件内容，并标记行号：grep -n 搜索的内容 文件名
  4. 搜索文件内容，反向查找：grep -v 搜索的内容 文件名
  5. 搜索以xx开头的文件内容：grep ^搜索的内容 文件名
  6. 搜索以xx结尾的文件内容：grep 搜索的内容$ 文件名

4. | - 管道符

  • 命令格式：命令1 | 命令2
  • “|”管道符的作用是把命令1的正确输出作为命令2的操作对象
  • 一般配合more使用
  1. 分页查看抓取到的内容：grep -v ^'#' /etc/services | more
    1. grep：抓取内容
    2. -v：取反
    3. ^’#’：以#开头
    4. more：分页打印
  2. 筛选输出的内容：ls -lh | grep 125

6. 文本输出

1. echo - 在控制台进行打印

2.>- 将结果保存起来

  1. 将文本输入到文件中，覆盖文件原有内容：echo hello > test01.txt
  2. 将文本输入到文件中，不覆盖文件原有内容：echo hello >> test01.txt
  3. 将列表信息，写入到文件中：ls > test01.txt
  4. 将查找的信息，写入到文件中：grep siki 126.txt >> test01.txt
    1. 将126.txt文件中，包含siki元素的行，写入test01.txt文件中
