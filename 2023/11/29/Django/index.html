四、请求、响应、返回值

  1. return HttpResponse：返回具体内容，将字符串返回给请求者

  2. return render(request,'someting.html',{"title":"xxx"})：读取HTML中内容 + 渲染（替换），生成字符串并返回给用户浏览器

  3. return redirect("http://xxx.xxx")：让浏览器重定向到其他页面

五、用户登录案例

  1. 用户get请求访问页面，浏览器post请求返回账户和密码

  2. 如果账户密码正确，则重定向到新的url

  3. 如果不正确，则返回html + 额外渲染的内容

  4. 需要在html的form表单中添加{% csrf_token %}，否则会出现csrf缺失

17. 后端生成图片，不保存图片，直接传递到前端界面

  • 将图像转换为Base64编码的字符串，然后在Django中使用HTML的img标签直接引用该字符串

  1. 要将 plt.show() 生成的图片传递给前端界面，可以使用 io.BytesIO() 和 base64 模块将图片转换为字节流和 base64 编码的字符串，然后将其传递给前端界面。

  2. 在这个示例中，我们使用 plt.savefig() 将图片保存为字节流，然后使用 base64.b64encode() 将字节流编码为 base64 字符串。最后，我们可以将这个字符串传递给前端界面，例如将其作为HTML的 <img> 标签的 src 属性值。
