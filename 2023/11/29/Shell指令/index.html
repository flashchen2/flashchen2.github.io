Shell指令

一、MacOS

1. 高频命令

  1. ls 显示当前文件夹下所有的文件
    ls -al 显示当前页面下的所有文件，可用于检测首页面所有隐藏文件夹，如配置环境变量的最终位置.bash_profile
  2. open .bash_profile 打开.bash_profile文件夹
    export PATH={PATH}:/文件路径（这是是mysql路径）/bin 配置路径
  3. cd /user/local/bin 进去mac配置环境的路径
    cd / 回到根目录
  4. mv chromedriver /user/local/bin 将chromedriver移动到/user/local/bin下，删除原位置文件
  5. ps -ef | grep mysql 全局查找“mysql”文件位置
    ls | grep mysql 当前路径查找“mysql”文件位置
  6. pwd 查看当前所在目录
  7. touch abcd 新建abcd文件
  8. rm abcd 删除abcd文件
    rm -rf abcd 强制删除abcd文件 （-r是递归处理，就是一层一层的删，-f是强制删除）
    sudo rm -drf 最高级别删除
  9. pip->pip3 因为使用的是python3
    pip3 install --upgrade Django==2.1 将Django版本升级为2.1版本
  10. which virtualenvwrapper.sh 查找virtualenvwrapper所在位置
  11. mkdir ~/pyEnv 创建一个pyEnv目录
  12. java -v 不支持
    java -version 支持

2. 查看本机ip

  • 注：个人Mac电脑对应得ip为10.220.164.103

3. 查找xx文件夹，并将路径输出到终端

4. 新建分屏终端页面

5. 取消分屏

6. MAC配置环境变量 ***

vi ~/.zshrc 进入配置环境变量
i&a 开始编辑，添加路径（i：在光标前追加；a：在光标后追加）
export PATH=${PATH}:/user/local/mysql/bin 以配置mysql为例
esc后，:wq! 退出当前添加
source ~/.zshrc 生效所配置的内容
mysql -u root -p 确认安装成功

2. 在终端中切换磁盘

3. 查找终端安装包的位置

  2. 查找某包的位置：pip show idna
  3. 可以直接将python中的包，复制到终端下

四、手表手环终端

1. 向固件中推入apk

  1. adb devices # pc是否通过usb检测到固件
  2. adb root
  3. adb remount
  4. adb push WearSysUi.apk /system/priv-app/WearSysUi/
  5. adb disable-verity
  6. adb shell reboot # 重启

2. 拉取固件日志

  1. adb bugreport #所拉取日志位于 此电脑-c盘-用户-用户处
