  1. for循环的过程可以通过上面的iter() 和 next()函数来实现
  2. for循环底层过程
    1. 先调用iter()，将nums当做实参，得到nums这个可迭代对象的迭代器
    2. 调用next(), 将上一步得到的迭代器进行取值
    3. 将上一步取出来的值，复制给num这个变量
    4. 执行for循环体中的代码，print(num)
    5. 接下来重复执行2/3/4步骤，当nums中的所有数据都获取完毕之后，会在下一次调用next的时候产生StopIteration异常

29. 闭包函数

1. 闭包基础

1. 介绍

  • 定义：闭包函数指的是定义在一个函数内部的函数，被外层函数包裹着，其特点是可以访问到外层函数中的名字

  • 优点：

    • 闭包可以保护函数内部的变量不被外部访问和修改，避免污染全局环境
    • 可以将局部变量永久存储，不会被垃圾回收机制回收

  • 缺点：数据长期驻留在内存中，会造成内存浪费，严重情况下会导致内存泄漏

2. 特点

  1. 内、外两层函数之间拥有嵌套关系

  2. 内层的函数使用了外层定义的变量

  3. 返回值是内层的函数体对象，且没有括号

  • 正常来说，调用一个函数的时候，这个函数中的所有局部变量+形参，都只会在这个函数执行的过程中才会保留，只要这个函数执行完毕，那么这些局部变量+形参，就会被自动释放
  • 闭包比普通的函数强大的地方在于，它在外部函数执行完毕之后，这个外部函数中的所有局部变量+形参都不会释放，以便于在调用内部函数的时候可以使用

30. 装饰器

  • 功能：能够在不修改代码的前提下，给函数添加功能，只能在原函数运行之前或之后添加，不能在运行一半时添加
  • 装饰器本质是一个闭包函数，是一个返回函数的函数

    • 装饰器：内部函数将外部函数中的变量，当做函数来调用
