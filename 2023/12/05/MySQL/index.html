二、 MySQL基础语句

1. 通用语法

  1. SQL语句可以单行或多行书写，以分号结尾
  2. SQL语句可以使用空格/缩进，来增强语句的可读性
  3. MYSQL数据库的SQL语句不区分大小写，关键字建议使用大写
  4. 注释：
    1. 单行注释：– 注释内容 或 # 注释内容
    2. 多行注释：/* 注释内容 */

2. DDL - 数据库操作

  1. 查询
    1. 查询所有数据库：SHOW DATABASES;
    2. 查询当前数据库：SELECT DATABASE();
  2. 创建数据库：CREATE DATABASE 数据库名称
  3. 删除数据库：DROP DATABASE 数据库名称
  4. 使用数据库：USE 数据库名称

3. DDL - 数据表操作

  1. 查询
    1. 查询当前数据库所有表：SHOW TABLES;
    2. 查询表结构：DESC 表名;
      1. DESC是describe的缩写，用于获取表的结构信息。
    3. 查询指定表的建表语句：SHOW CREATE TABLE 表名;
  2. 创建
  • 创建表中最后一个字段后面没有逗号
  1. 示例1：
id name age gender
1 令狐冲 28 男
2 风清扬 68 男
3 东方不败 32 男

  1. 示例2：

    设计一张员工信息表，要求如下：

    1. 编号（纯数字）
    2. 员工工号（字符串类型，长度不超过10位）
    3. 员工姓名（字符串类型，长度不超过10位）
    4. 性别（男/女，存储一个汉字）
    5. 年龄（正常人年龄，不可能存储负数）
    6. 身份证号（18位数，结尾可能是x）
    7. 入职时间（取值年月日即可）

三、 高频应用

0. mysql的启动和关闭

  1. 启动服务器：net start mysql
  2. 服务停止命令：net stop mysql
  3. 退出mysql：exit；
  4. 终止此行语句：ctrl + c

1. 进入mysql

  1. 访问mysql：mysql -u root -p
  2. 重置密码：set password = password(“Pooh_0505”)

2. 对mysql中文件夹操作

  1. 查看mysql目录下文件夹：show databases;
  2. 创建数据库：create database 数据库名字 DEFAULT CHARSET utf8 COLLATE utf8_general_ci;
  3. 删除数据库：drop database 数据库名字;
  4. 进入数据库：use 数据库名字;
  5. 查看文件夹下所有的数据表：show tables;

3. 对具体文件操作

  1. 创建表：create table 表的名称（列名称 类型，列名称 类型，列名称 类型）default charset=utf8;
  2. 删除表：drop table 表的名称;
  3. 查看表的格式：desc 表的名称;
  4. 插入数据：insert into 表的名称（列1名称，列2名称） values (列1数据，列2数据），(列1数据，列2数据）;
  5. 查看表中的数据：select * from 表的名称;

注： 主键一般用于表示当前行的数据的编号 最后一行列表不能加'，'

4. 数据行操作

  1. 新增数据：insert into 表的名称（列1名称，列2名称） values(列1数据，列2数据），(列1数据，列2数据），(列1数据，列2数据）;
