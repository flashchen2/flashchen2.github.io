二、 MySQL的启动和关闭

  1. 启动服务器：net start mysql
  2. 服务停止命令：net stop mysql
  3. 退出mysql：exit；
  4. 终止此行语句：ctrl + c

三、 高频指令

1. 进入mysql

  1. 访问mysql：mysql -u root -p
  2. 重置密码：set password = password(“Pooh_0505”)

2. 对mysql中文件夹操作

  1. 查看mysql目录下文件夹：show databases;
  2. 创建数据库：create database 数据库名字 DEFAULT CHARSET utf8 COLLATE utf8_general_ci;
  3. 删除数据库：drop database 数据库名字;
  4. 进入数据库：use 数据库名字;
  5. 查看文件夹下所有的数据表：show tables;

3. 对具体文件操作

  1. 创建表：create table 表的名称（列名称 类型，列名称 类型，列名称 类型）default charset=utf8;
  2. 删除表：drop table 表的名称;
  3. 查看表的格式：desc 表的名称;
  4. 插入数据：insert into 表的名称（列1名称，列2名称） values (列1数据，列2数据），(列1数据，列2数据）;
  5. 查看表中的数据：select * from 表的名称;

注： 主键一般用于表示当前行的数据的编号 最后一行列表不能加'，'

4. 数据行操作

  1. 新增数据：insert into 表的名称（列1名称，列2名称） values(列1数据，列2数据），(列1数据，列2数据），(列1数据，列2数据）;

5. 其他指令

  1. 显示目前已有的用户名：select user from mysql.user;
  • 用户名默认是root
  1. 查看当前mysql数据库的版本：select version();

四、常用的数据类型

  1. 整型：tinyint & int & bigint

  2. 有符号范围：-128~127 【默认】
    age tinyint

  3. 无符号范围：0~255
    age tinyint unsigned

  4. 小数：float & double & decimal

  5. salary decimal(8,2)
    其中salary总共有8位数，小数点前6位，小数点后2位

  6. 字符串：char & varchar & text

  7. char 定长字符串
    mobile char(11)
    固定使用11个字符串进行存储，哪怕没有11个

  8. varchar变长字符串
    mobile varchar(11)
    真实数据有多长，就会存储多长

  9. 时间：datetime & time

  10. datetime：YY-MM-DD HH:MM:SS

  11. time：YY-MM-DD
